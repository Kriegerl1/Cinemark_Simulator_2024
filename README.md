Controle de Cinema

Controle de Cinema é um projeto desenvolvido em C# que visa criar uma aplicação para gerenciar cinemas, incluindo a administração de filmes, sessões e ingressos. O projeto está organizado em várias camadas, incluindo domínio, infraestrutura e uma aplicação web.
Estrutura do Projeto

O projeto está dividido em várias pastas principais:

    Apresentação: Contém a interface do usuário.
    Domínio: Inclui as regras de negócio.
    Infraestrutura: Responsável pelo acesso a dados e serviços externos.
    Testes: Contém testes de integração e unitários para validar o funcionamento do sistema.

Requisitos

    .NET Core SDK
    Visual Studio 2022 (ou versão mais recente)

Instalação

Para rodar o projeto localmente, siga os passos abaixo:

bash

# Clone o repositório
git clone https://github.com/POG-Syntax/Gerenciamento-de-Cinema-2024

# Navegue até o diretório do projeto
cd Controle-de-Cinema

# Abra o projeto no Visual Studio e compile
